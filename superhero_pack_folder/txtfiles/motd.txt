say commands:
/general_help -> show this again

/help -> superheromod help

note: use lowercase when binding a power to a letter key
examples:
bind i +power2
(use "i", not "I".)

/extra_help -> more superheromod help

/helpof [part of hero name] -> get help about hero
		(default file will be shown
		if a specific one does not exist)

console:
sh_print_models -> show a list
		of pickable player models
		avaliable to you
		based on currently equipped heroes

sh_choose_model [model id] -> pick one model
		from the model_id values listed
		by the previous command
			
sh_print_weapon_models -> show a list
		of pickable weapon models
		avaliable to you
		based on currently equipped heroes.
		Weapon IDs and model IDs for each weapon
		will be shown along with hero name

sh_choose_weapon_model [weapon id] [model id] ->
		pick model of id [model_id]
		of weapon [weapon_id]
		valid values are listed
		by the previous command
