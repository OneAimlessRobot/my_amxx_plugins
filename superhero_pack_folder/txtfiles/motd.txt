Say commands:
/help -> superheromod help
by the way: use lowercase when binding a power to a letter key
Like v (not V) or e (not E)
examples:
power2: bind i +power2
(use "i".
NOT "I".)
power1: bind q +power1
(use "q".
NOT "Q".)

/extra_help -> more superheromod help

/helpof [part of hero name] -> get help about hero
			(default file will be shown
			if a specific one does not exist)

Console commands:
sh_print_models -> show a list
			of pickable player models
			avaliable to you
			based on currently equipped heroes

sh_choose_model [model id] -> pick one model
			from those listed
			by the previous command
			(if you still have
			said required hero equipped)
			
If you wish to see this file again
Use the say command: /general_help
