this hero has a custom player model!

passive abilities:

Three custom weapons:
1- The pillgatling.
The pillgatling is a minigun.
It shoots pills that are quite lightweight
and therefore drift chaotically in the wind
as they fly through the air.

2- The syringe rocket launcher:
It shoots rockets.

Now.

both of these weapons use the same platform
a M249 given to the user
And one can switch between them by pressing the ALT1 key
(bind "key" +alt1 to use if not bound)
while having that weapon deployed

On pillgatling mode,
the pills are shot through Attack1
(mouse left click by default)
And in rocket mode,
rockets are shot with attack2
(mouse right click by default)

And the special thing about both of these weapons
is that they can apply custom effects to users.

the rockets will apply them within blast radius
and pills will apply them on hit
And these effects will all (currently)
Last a similar amount of time
and run at second long periods.
