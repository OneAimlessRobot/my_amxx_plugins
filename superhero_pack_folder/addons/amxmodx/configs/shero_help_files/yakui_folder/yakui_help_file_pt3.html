Effects list:

- Cyanide -> 
Color: Cyan
Desc: this is a special one.
it does not really do anything
other than instantly kill
anyone who gets hit with it.
Chance: 1%

- Glow ->
Color: white
Desc: the most common effect.
it simply makes the user glow
Chance: 10%

- Poison ->
Color: Green
Desc: This effect
causes periodic damage to be dealt.
But it also increases
all hitscan damage
dealt by the user by a factor of two.
Poison damage taken by someone who is poisoned
will double if also bleeding
It makes the user glow too
Chance: 10%

Stunner ->
Color: Light grey
Desc: It stuns the user
And makes them glow
Chance: 10%

Radioactive ->
Color: Orange
Desc: It deals periodic damage,
glows the user
And makes them show up
in the enemy radar.
Enemies also get tracers
pointing to where the user is
The damage is substantial.
Chance: 10%

Morphine ->
Color: Light Olive green
Desc: It periodically heals the user
Chance: 10%

Weed ->
Color: Faded jade green
Desc: It sets the user's gravity to 10%
Chance: 11%

Cocaine ->
Color: Red
Desc: It sets the user's max speed to 1000
Makes them shoot and reload faster
But it also causes bleeding.
It makes the victim glow as well.
Chance: 8%

Blindness ->
Color: White
Desc: It colors the user's screen
to pitch white
making it impossible to see
Chance: 6%

Methylphenidate ->
(I probably mispronounced it)
Color: Shallow grey
Desc: it nullifies all shooting recoil
Chance: 5%

Bath salts ->
Color: Orange
Desc: it nullifies the usual knockback
dealt by hitscan weapons
Chance: 10%
